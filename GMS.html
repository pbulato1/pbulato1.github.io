About

GMS (Gym Management System) is a record management system designed for use by a gym's authorized staff members. It contains an authentication system which prompts the user for the username and password.Following a successful log-in, the user is presented with the Main Menu, which contains several features. Some of them include adding new members and equipment, or registering staff. Also, the user can access the existing data from the database, as well as modify it according to their needs. The program was implemented in C# and SQL.
